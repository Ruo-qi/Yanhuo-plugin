# 安装使用

```bash
$ pnpm install --save yanhuo-hooks
or
$ yarn add yanhuo-hooks
```

# CDN

目前可以通过 **unpkg.com/yanhuo-hooks** 获取到最新版本的资源，在页面上引入即可开始使用。

```bash
<script src="https://unpkg.com/yanhuo-hooks"></script>
```
